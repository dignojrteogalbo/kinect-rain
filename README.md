# kinect-rain

## Getting Started

### Required Dependencies

Python

```
sudo apt-get install python3
```

Tkinter

```
sudo apt-get install python3-tk
```

OpenKinect

```
sudo apt-get install freenect
```

libusb
```
sudo apt-get install libusb-dev
```

CMake
